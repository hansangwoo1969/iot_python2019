주소록

주소록.txt 파일에는 이름 주소(대도시(도), 구(군)까지 존재 대도시와 군사이 공백 한칸)가 한줄에 있고 이름과 주소 사이 탭이 한개존재
ex)
손병찬\t대구광역시 남구
홍길동\t서울특별시 강남구
제갈공명\t전라남도 해남군

프로그램 시작시 환영인사 메세지 출력
모드 선택 "1. 검색 모드 2. 추가 입력모드(종료 = 0) " 메세지 출력 후 0~2 입력받음
1 입력시 검색모드로 이동 2 입력시 추가 입력모드로 이동 0 입력시 프로그램 종료

1. 검색모드:
"1. 이름 검색모드 2. 주소검색모드 (종료= 0)" 메세지 출력후 0~2 입력받음
1 입력시 이름 검색모드로 이동 2 입력시 주소검색모드로 이동 0 입력시 검색모드 종료

-이름검색모드-
이름을 입력 받으면 해당하는 이름과 주소를 출력함
이름이 없으면 해당하는 이름이 없습니다 메세지 출력 후 검색모드 처음으로 이동

-주소 검색모드-
주소는 대도시(한단어)만 검색하여 대도시입력시 대도시에 속한
사람의 이름과 주소를 출력함
해당하는 주소가 없으면 해당하는 주소가 없습니다 메세지 출력후 검색모드 처음으로 이동


2. 추가 입력모드 선택시 추가 입력 모드를 선택하였습니다 메세지 출력 후 이름을 입력 받음

-입력된 이름이 주소록에 있으면 입력된 이름과 주소를 출력하고 난뒤 메세지 출력 "1.수정 2.종료" 메세지 출력후
입력 받음  2 입력시 추가 입력모드 종료되었습니다 메세지 출력후 프로그램 맨 처음으로 이동

1 입력시 수정모드 입니다 메세지 출력후 주소 입력받음

-주소록에 이름이 없으면 추가 주소 입력하라는 메세지와 함께 주소 입력받음

--수정하거나 추가 하는 주소입력이 끝나면 이름과 주소 출력하여 이름과 주소가 맞는지 확인받음
1. 확인 2. 재입력
재입력 선택지를 선택시 추가 입력모드 처음 이름 입력받는 곳으로 이동

확인선택시 이름과 주소를 주소록.txt 에 저장하고 프로그램 맨 처음으로 이동함
